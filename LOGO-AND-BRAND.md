## If You Want to Change Colors

**Super easy** - just edit the top of `styles.css`:

```css
:root {
    /* Change these two lines: */
    --primary-color: #2C5F6F;  /* Your new primary color */
    --accent-color: #7AB8A8;   /* Your new accent color */
}
```

Everything else updates automatically!

## Using Your Own Logo

If you decide to use a different logo:

### What I Need:
1. **File format**: SVG (best), PNG (acceptable), or high-res JPEG
2. **Transparent background** if possible
3. **Orientation**: Horizontal works best (logo + text side by side)
4. **Size**: At least 500px wide

### How to Replace:
1. Save your logo as `logo.png` (or `logo.svg`)
2. Put it in the same folder as the HTML files
3. In each HTML file, replace the `<svg class="logo">` section with:
   ```html
   <img src="logo.png" alt="Rejuvenate Health MKE" class="logo">
   ```
4. Done!

## Brand Voice (Based on Your Copy)

Your brand personality should be:

### ✅ Do:
- Professional but personable
- Educational without being condescending
- Confident but not arrogant
- Empathetic to concerns
- Clear and jargon-free when possible

### ❌ Don't:
- Too clinical/cold
- Over-promising results
- Using scare tactics
- Being too "sales-y"
- Excessive medical jargon

### Example Tone:
**Good**: "Emsculpt NEO builds muscle and burns fat simultaneously - think of it as thousands of crunches in 30 minutes, but more effective."

**Too Clinical**: "Utilizing HIFEM+ technology, the device induces supramaximal contractions resulting in hypertrophy and lipolysis."

**Too Sales-y**: "GET THE BODY YOU'VE ALWAYS DREAMED OF! LIMITED TIME OFFER!"

## Logo Don'ts (Avoid These Mistakes)

❌ **Don't use clipart** - looks amateur
❌ **Don't use too many colors** - gets messy
❌ **Don't use trendy fonts** - will look dated quickly
❌ **Don't make it too complex** - won't scale down
❌ **Don't ignore mobile** - logo must work tiny

✅ **Your current logo avoids all these problems!**

## Questions to Ask Yourself

1. **Do I want to be seen as more medical or more spa-like?**
   - More medical → Keep it simple, use navy heavily
   - More spa → Emphasize teal, softer shapes

2. **Who's my primary audience?**
   - Aesthetic clients → Approachable, modern
   - Rehab clients → Professional, trustworthy
   - Both → Balanced (which is what we have now)

3. **What's my competition doing?**
   - Look at other Emsculpt providers in Milwaukee
   - Don't copy them, but differentiate

4. **Does this feel like ME?**
   - Your brand should feel authentic to you
   - If something feels off, let's change it
